# Local — Your Sovereign Space

This folder belongs to your fork. **Upstream will never touch it.**

When you merge updates from `commons-engineering/commons-os`, everything in `_local/` stays exactly as you left it. This is by design.

## Structure

| Folder | Purpose |
|---|---|
| `_patterns/` | Patterns specific to your instance |
| `_lighthouses/` | Lighthouse briefings for your stakeholders |
| `_config/` | Instance-specific configuration |

## Convention

Put anything instance-specific here. The rest of the repo may be updated by upstream PRs — this folder is yours alone.
