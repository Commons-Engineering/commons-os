01 // THE DIAGNOSIS

What Is a Commons?

Canonical Definition
A commons systematically creates value that is alive, shared, just, and built to last.
Quality
Meaning
Test Question
Systematically
Through design, architecture, and patterns, not individual heroics.
Is this repeatable?
Alive
Regenerates through use. Grows, adapts, and dies when untended.
Does this make the system more vital?
Shared
Belongs to its participants collectively. Neither private nor state.
Does value serve all who enable it?
Just
Fair. Pervades all dimensions. Prevents Enclosure as Code.
Does this serve all fairly?
Built to last
Outlasts its founders. Stewardship rotates. Knowledge is documented.
Would this survive without us?
