Commons as Code

Every Commons is a lens on any type of value creation — of any piece of reality or fiction. Because it is built as code, you can merge entire pathways into a single context window.

THE LENS
A Commons models a specific view of value creation — reality or fiction. Model a hospital as it is today, or as it could be in 2035. Model a city district, an ecological zone, or a "maybe reality." As many views as desired.
AS CODE
Because these models are written as code (not just static documents), they are fully machine-readable, composable, and interoperable.
THE CONTEXT WINDOW
Merge reality and fiction into a single AI context window. Your present state, your projections, your maybe-realities — entire pathways side by side. The AI gains a multi-dimensional understanding across time, domain, and possibility.
